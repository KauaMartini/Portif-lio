Netflix: Meu App de Filmes e Séries

Este projeto é um aplicativo de filmes e séries onde os usuários podem navegar por diferentes categorias e assistir trailers de filmes. A interface foi construída usando HTML, CSS e JavaScript para criar uma experiência interativa. Além disso, a busca por filmes é feita através de uma API de filmes, o que permite que as informações dos filmes sejam constantemente atualizadas.
